# Relocation Guide

Платформа для переезда из России в другие страны.

**Сайт:** https://unfoldqa.github.io/relocation-county/

## Возможности

- **26 стран** с описаниями, плюсами/минусами, рейтингами и ссылками на официальные источники
- **Фильтрация** по региону, визе, бюджету, климату, nomad-friendly
- **5 интерактивных тестов**: подбор страны, барьеры, готовность, «почему не могу», стиль жизни
- **Сравнение** до 3 стран side-by-side
- **Калькулятор бюджета** с настройкой уровня комфорта
- **Roadmap** из 12 шагов с отслеживанием прогресса
- **20+ Telegram-каналов** и полезных сайтов
- **Геймификация**: достижения, избранное, confetti-эффекты
